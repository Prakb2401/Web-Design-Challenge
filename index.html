Weather API Data Analysis

Summary: Latitude vs X

In this challenge I used API from open weatherAPI to get, request information, and create visualizations with it. I used the OpenWeatherAPI to get weather information of over 500 cities. After that I created a dataframe with that and outputted a csv file. I then created 4 scatter plots to show correlations based on latitude and other weather conditions such as temperature, wind speed, humidity, and cloudiness.
